
# Valbruna MM BTP - Repository Principale

## Descrizione

Questo repository raccoglie la soluzione Valbruna MM sviluppata su SAP Business Technology Platform (BTP), comprendente le applicazioni **RdA** (Richieste di Acquisto) e **Magazzino Ausiliario**. Il progetto integra backend CAP, frontend SAPUI5/Fiori Elements e automazione infrastrutturale tramite Terraform, per la gestione, monitoraggio e automazione dei processi di richiesta, approvazione e tracciamento materiali.
